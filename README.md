#COMIDA POR DELIVERY

Pseudocódigo y diagrama de flujo para un pedido de comida por delivery.

1. Inicio 
2. Decidir que comida comprar
3. Llamar al restaurante de preferencia
4. Consultar precios u ofertas 
5. Brindar datos
6. Recepción de comida
7. Pago
8. Fin
